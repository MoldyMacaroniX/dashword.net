Geode, the most popular Geometry Dash modloader, is now available for [version 2.208](/posts/geometry-dash-2-208-released/) after a nearly month long wait. Mods are still being updated but Geode is now available to download and install again.

Geode is the mod loader used by popular mods like [Globed](/posts/geometry-dash-multiplayer-how-to-download-and-install/) and [Click Between Frames](/posts/geometry-dash-click-between-frames-mod-download-and-install/).

However, Geometry Dash 2.208 adds support for Click On Steps, which is the new official version of Click Between Frames, so it is unclear if that mod will be updated to work in the latest version.

# Download and Install Geode for Geometry Dash 2.208

While originally only available on their [GitHub Repo](https://github.com/geode-sdk/geode/releases), you can now download and install Geode directly from the Geode website!

1. [Backup your save data](/posts/backup-gd-data-steam/). This is always a good idea when modding anything!
2. Download the Geode Installer for your device from [the official Geode website](https://geode-sdk.org/install). Version 5 is the latest version of Geode, an upgrade from the previous version 4.
3. Run the installer.
4. Open Geometry Dash and download mods using the in-game Geode button!

# Why did Geode take so long to update?

The Geode developers have confirmed that Geode for 2.208 took so long to make because they were implementing [a lot of internal changes](https://youtu.be/BT91PCUdyOU?si=oBybsDzGM3_Cu7dL) to make it more optimized.

Fortunately, this means Geode should run a lot better now!

It is also worth noting that future updates to Geode should not take this long since the team has finished making internal changes for this release.
